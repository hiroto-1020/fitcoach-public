# FitGear（公開用）

重要ポイントのみまとめています。

---

### React周り

- `app/_layout.tsx`：アプリの入口（ルートレイアウト）
- `app/(tabs)/home/index.tsx`：代表画面の実装例
- `lib/`：ロジック・データアクセスまわり
- `supabase/functions/`：Functions（サーバー側相当）の実装

---

### ざっくり構成

- `app/`：画面・ルーティング
- `components/`：共通コンポーネント
- `ui/`：UI部品・テーマ
- `lib/`：ロジック・データアクセス
- `hooks/`：カスタムフック
- `locales/`：多言語
- `supabase/functions/`：Functions 実装

---

### 補足

- 公開用のため、設定値（URL/キー等）はリポジトリに実値を入れていません。
- こちらのリポジトリ自体実際のものをクローン化しており、非公開部分はすべてサンプルとさせていただきました。
